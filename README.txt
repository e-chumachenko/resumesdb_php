Веб-приложение по созданию, редактированию, удалению, хранению и поиску резюме

Языки: HTML5, CSS3, JavaScript (jQuery), PHP5, SQL
Технологии: PDO для доступа к базе данных, AJAX для ввода с автодополнением
Среда разработки: XAMPP, MAMP, MySQL, Notepad++ на Windows

Веб-приложение состоит из 11 файлов:

- index.php
    + выводит на экран список резюме в виде таблицы
    + есть опция поиска по базе данных резюме
    + названия должностей в таблице являются гиперссылками для просмотра резюме
    + для верифицированных пользователей есть опции создания, редактирования, удаления своих резюме и выхода из своей учётной записи
    + для неверифицированных пользователей есть ссылки на страницы регистрации и входа в свою учётную запись 
    
- login.php 
    + содержит поля формы для ввода адреса электронной почты и пароля пользователя
    + удаляет из сессии данные предыдущего пользователя
    + проверяет заполнение полей и наличие @ в адресе электронной почты
    + если пользователь с такими регистрационными данными существует в базе данных, сохраняет в сессию его id, эл. почту, имя и фамилию
    + перенаправляет пользователя на index.php
    
- registry.php
    + содержит поля формы для ввода адреса электронной почты, пароля и подтверждения пароля пользователя
    + удаляет из сессии данные предыдущего пользователя
    + проверяет заполнение полей, наличие @ в адресе электронной почты и равенство пароля и подтверждения пароля пользователя
    + если эл. почта уже есть в базе данных, перенаправляет пользователя на login.php c соответствующим флеш-сообщением
    + добавляет эл. почту и пароль в базу данных (пароли хранятся в базе данных с использованием соль-криптографии)
    + сохраняет в сессию id и эл. почту пользователя
    + перенаправляет пользователя на index.php
    
- view.php
    + проверяет, существует ли запрашиваемое пользователем резюме
    + выводит на экран резюме, выбранное пользователем

- add.php
    + содержит поля формы для создания резюме
    + при нажатии на "+"/"-" JavaScript(jQuery) создаёт/удаляет дополнительные поля формы об образовании и опыте работы
    + поле "ВУЗ" содержит опцию ввода с автодополнением (jQuery), варианты автодополнения запрашиваются у сервера school.php методом AJAX 
    + проверяет заполнение полей, отмеченных *
    + добавляет резюме, информацию об образовании и опыте работы пользователя в соответствующие таблицы базы данных
    
- edit.php
    + проверяет, верифицирован ли пользователь, существует ли запрашиваемое им резюме, и принадлежит ли оно ему
    + выводит на экран поля формы, заполненные информацией пользователя из базы данных 
    + проверяет заполнение полей, отмеченных *
    + обновляет данные резюме, а также удаляет прошлые и вводит новые данные об образовании и опыте работы пользователя

- delete.php
    + проверяет, верифицирован ли пользователь, существует ли запрашиваемое им резюме, и принадлежит ли оно ему
    + методом confirm() JavaScript повторно подтверждает намерение пользователя удалить своё резюме
    + удаляет резюме пользователя из базы данных

- school.php 
    + cоздаёт массив в формате json с названиями ВУЗов, содержащими вводимую пользователем комбинацию букв (необходим для работы 
      ввода с автодополнением (jQuery) поля "ВУЗ" файлов add.php и edit.php)

- logout.php
    + удаляет из сессии данные пользователя
    + перенаправляет пользователя на index.php

- pdo.php 
    + осуществляет доступ к базе данных MySQL при помощи PDO и устанавливает его атрибуты

- stl.css
    + создаёт адаптивный дизайн с использованием относительных измерений и медиа запросов
    
Презентация 'PHP.pptx' содержит снимки экранов index.php, login.php, registry.php, view.php, add.php, edit.php, delete.php, 
а также модель базы данных веб-приложения.

Файл 'SQL для создания БД.txt' содержит полный список SQL запросов для создания базы данных, необходимой для работы приложения.
